# 聊天存檔說明

本資料夾用於存放**與此子專案相關的聊天記錄**，方便之後在新聊天中說「調用過去的聊天記錄」時，AI 能讀取到脈絡。

---

## 為什麼要存檔

- 每次研究你會**開新的聊天窗口**，舊對話不會自動帶過去。
- 若要**延續或修改**先前結論（例如改止損、換參數、加過濾條件），需要讓 AI 知道當時做過什麼決定。
- 把關鍵聊天**存到本子專案的 chat-archive**，之後用「@ 檔案」或「調用 btc-4h-macd-divergence 的聊天記錄」即可引用。

---

## 如何存檔

### 方式一：匯出對話後存成檔案（推薦）

1. 在 Cursor / Claude Code 中將該次聊天**匯出**（若有「Export chat」或「Save transcript」）。
2. 把匯出內容存到本資料夾，檔名建議：
   - `YYYY-MM-DD_簡短描述.md`  
   - 例：`2025-02-25_初版回測與止損修正.md`
3. 在 **`INDEX.md`** 加一筆，寫明日期、檔名、一句話摘要（方便之後搜尋）。

### 方式二：手動摘要

若沒有匯出功能，可自己建一個 `.md`，內容包含：

- 該次聊天做了哪些事（例如：建立回測、止損改為底背離低點、跑出 43 筆）
- 重要結論或參數（勝率、止盈止損設定、時間止損）
- 之後若要改動，可從哪裡改（config、哪個檔）

檔名同樣建議 `YYYY-MM-DD_描述.md`，並在 `INDEX.md` 登記。

---

## 如何讓 AI 調用過去的記錄

在新聊天中可說：

- 「請先讀 `projects/btc-4h-macd-divergence/chat-archive/` 裡的存檔，再幫我改止損為前一根 K 低點。」
- 或直接 **@ 某個存檔檔案**（例如 `@chat-archive/2025-02-25_初版回測.md`），再下指令。

AI 會依你指定的存檔與當前需求，延續當時的設定與邏輯進行修改。

---

## INDEX.md

請在 **`INDEX.md`** 維護一個簡單索引，例如：

```markdown
| 日期       | 檔名                    | 摘要           |
|------------|-------------------------|----------------|
| 2025-02-25 | 2025-02-25_初版回測.md  | 建立策略、止損改為背離低點、43筆回測 |
```

之後要「調用過去的聊天記錄」時，可先看 INDEX 再決定 @ 哪一個檔案。
